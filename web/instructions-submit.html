Instructions

This is a short descriptions of the fields, most directly copied from the FUNC website. Please check the FUNC documentation if you want to know more about the FUNC tool.

.
Project Name: This is an arbitrary string to describe your project. You might want to enter a meaningful name to distinguish between several submissions.
e-mail address: An email will be sent to you after you submitted the form. The email contains the link to check the status of your analysis and fetch the results.
Input File: Upload your tab-separated input file here. You can find a detailed description on the format here
Test: Choose which test should be used to analyse your data. See section methods here for a full description how the p-values are calculated.
Ontology: The Ontology over which to perform the analysis. We download the latest ontology versions in regular intervals (3 months) and do not currently keep older ontology versions.
Cutoff for number of genes/group: Only categories with at least cutoff annotations will be included in the analysis.
Number of random sets: How many random sets should be created to asses the corrected p-values and the overall significance?
Root nodes: Input the identifier of the ontology class that will serve as root node in the analysis. owl:Thing is the root node of any ontology class. This option is useful when classes from multiple ontologies are included in an ontology and you would like to perform the analysis only over one particular type of class.

Data handling policy

All input data is deleted once a job is completed, either successfully or unsuccessfully (as a result of an error). The results are kept for at least one year. If you would like us to delete your analysis results for any reasons, please send an email to Robert and state your job id.
